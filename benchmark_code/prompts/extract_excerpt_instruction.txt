You are a research assistant specialized in extracting key information from educational studies.

PHASE 2 OF 2: INSTRUCTIONAL EVIDENCE EXTRACTION WITH LINKING
Your goal is to extract ALL verbatim evidence about TUTOR/TEACHER ACTIONS and LINK each to relevant learner behaviors.

────────────────────────────────
### CRITICAL RULES
────────────────────────────────
1. Extract EXTENSIVELY - MAXIMUM COVERAGE of instructional evidence
2. ALL text MUST be VERBATIM from the document - NO paraphrasing, NO summarizing, NO inference
3. EVERY excerpt MUST include source location (page/section)
4. LINK each instructional excerpt to relevant learner excerpt(s) from PHASE 1
5. Output ONLY valid JSON - NO explanations, NO markdown


────────────────────────────────
### INPUT CONTEXT
────────────────────────────────
ALL STUDENT CATEGORIES:
{CATEGORIES}

LEARNER EXCERPTS from PHASE 1:
{LEARNER_EXCERPTS}

- Use "name" and "label" field to identify which category_id(s) each excerpt refers to.
- Your task is to extract evidence for ALL of them.


────────────────────────────────
### TASK OVERVIEW
────────────────────────────────

**STEP 1: Scan and Tag**
Read the entire paper and mentally note ALL locations containing TEACHER-CENTERED content:
  - Effective teacher actions (strategies that worked, recommended practices)
  - Ineffective teacher actions (strategies that failed or actions to be avoided)
  - Intervention descriptions (what teachers DID or SHOULD DO)
  - Outcomes (student responses to interventions - as results of teacher actions)

DO NOT tag STUDENT-CENTERED content already covered in PHASE 1:
  - Pure behavioral observations without intervention context → Skip for PHASE 2

**STEP 2: Extract Evidence**
For EACH tagged location:
  - Copy excerpt with 2-3 surrounding sentences VERBATIM (word-for-word from paper)
  - Focus on passages that primarily describe what TEACHERS did or should do
  - Identify applicable category_id(s)
  - Classify excerpt_type: tutor_action_effective | tutor_action_ineffective
  - Fill instructional_context:
    * when: Conditions when this applies (e.g., "when student shows frustration")
    * how: Concrete steps/approach (e.g., "pause activity, ask guiding question, provide visual")
    * outcome: Student response/result (e.g., "student successfully solved similar problem")
  - Classify temporal_scope:
    * immediate: Applied in-the-moment (e.g., "Ask guiding question", "Provide worked example")
    * long_term: Sustained over time (e.g., "Establish error-friendly culture", "Gradually increase difficulty")
    * both: Has immediate AND sustained components

**STEP 3: Link to Learner Behaviors**
For EACH instructional excerpt:
  - Review learner excerpts from PHASE 1
  - Identify which learner behaviors this instruction addresses
  - Write explicit linking_rationale with evidence from paper
  - Assign link_strength:
    * 3 (Strong): Paper explicitly connects instruction to this behavior
    * 2 (Moderate): Addresses same challenge type; temporal scopes align
    * 1 (Weak): General strategy that could apply; same category
    * 0 (No link): Different behavior/category or temporal mismatch

**STEP 4: Verify Coverage**
Check for gaps:
  - Any orphaned learner excerpts (no linked instructions)? → Search for relevant strategies
  - Any category with <5 instructional excerpts? → Search harder
  - Any vague instructional_context? → Add concrete details from paper

**STEP 5: Verbatim Verification (CRITICAL)**
For EVERY excerpt, perform strict verification:
  1. Re-check EACH item against the source document
  2. Verify word-by-word match (including punctuation, capitalization)
  3. Record result in excerpt_verbatim_check dictionary
    - excerpt_verbatim_check: {"{ID}_instruction_001": true, "{ID}_instruction_002": false}
  4. If ANY item is not verbatim (paraphrased/summarized), mark as false
  5. Explain any non-verbatim items in verbatim_reasoning
