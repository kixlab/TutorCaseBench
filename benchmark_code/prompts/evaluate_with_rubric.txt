Score the tutor's response using the provided personalized rubric.

────────────────────────────────
### SCORING INSTRUCTIONS
────────────────────────────────

For each criterion in the rubric:
- Read the "check" condition carefully
- Score 1 if the tutor response satisfies the condition, 0 if not
- The check is principle-level: diverse approaches can earn the point as long as the underlying principle is present
- Be strict but fair — surface mentions without substance do NOT earn the point
- Do NOT reward length, politeness, or fluency

A tutor response does not need to satisfy every criterion to be considered acceptable; the rubric documents the full set of principles this student needs, and a single response can only address some of them. Score each criterion independently.
