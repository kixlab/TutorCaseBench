RULES
────────────────────────────────

PRESERVE
  - The MECHANISM of each option (e.g., "scaffold by chunking", "visual + verbal cue", "behavior-specific praise", "offer a binary choice"). Do not switch mechanisms.

CHANGE — surface only

LENGTH & TONE PARITY
  - Priority when rules conflict (higher wins):
      1. Each option's MECHANISM unchanged; the correct option stays right for THIS
         student; each distractor stays wrong for THIS student. NEVER violate this.
      2. The four options roughly similar in length and tone.
  - Equalize length by TRIMMING the longer options first.
  - If trimming alone cannot close the gap, you MAY add a small amount of NEUTRAL

    effect, benefit, or empirical/evidence support (no "...which improves
    performance", "...with strong research backing"); and MUST NOT make a

    If you cannot pad without touching the mechanism or its fit, leave it short —
    an uneven length is acceptable; changing substance is not.
  - All four in third-person tutor-action voice (never direct speech, never "you").
  - Match register across all four: professional, concise, action-focused.

MINIMUM EDIT
  - Swap only the surface terms the target context requires. No swap needed →
    output verbatim.
  - Never introduce a new method/technique/step, never flip or soften substance.
    Neutral length-matching filler (per LENGTH & TONE PARITY) is the ONLY permitted
    addition.

EDIT LEVELS (one label for the whole pass)
  - none    : all four output verbatim.
  - surface : ≤ 3 surface words swapped per option.
  - light   : at least one option needed a short phrase rewrite (no direct target analogue for a surface element).
  - fuller  : AVOID. Only if a setting structure had no target equivalent and had to be re-described. Re-check that all mechanisms are intact and no wrongness was softened.
  If any option received neutral length-matching filler, say so explicitly in
