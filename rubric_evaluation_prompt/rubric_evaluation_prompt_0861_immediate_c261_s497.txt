### SCORING INSTRUCTIONS
────────────────────────────────

For each criterion in the rubric:
- Read the "check" condition carefully
- Score 1 if the tutor response satisfies the condition, 0 if not

- Be strict but fair -- surface mentions without substance do NOT earn the point

A tutor response does not need to satisfy every criterion to be considered acceptable; the rubric documents the full set of principles this student needs, and a single response can only address some of them. Score each criterion independently.

### RUBRIC (variable-length checklist)
────────────────────────────────

Criterion 1: Physical Level Engagement
  Check: The tutor's response includes a physical action to enter, open, or approach the play space (e.g., kneeling, opening tent flaps, moving to their level) to deliver the intervention.

Criterion 2: Play Validation vs. Destruction Boundary
  Check: The tutor explicitly names and validates the acceptable imaginative goal (e.g., making a home, setting a table) while setting a firm verbal boundary against the destructive behavior (e.g., trashing, throwing).

Criterion 3: Concrete Consequence Explanation
  Check: The tutor provides a concrete, physical reason for the boundary (e.g., things get broken, it makes a big mess).
