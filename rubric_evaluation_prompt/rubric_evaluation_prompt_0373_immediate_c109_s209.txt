### SCORING INSTRUCTIONS
────────────────────────────────

For each criterion in the rubric:
- Read the "check" condition carefully
- Score 1 if the tutor response satisfies the condition, 0 if not
- The check is principle-level: diverse approaches can earn the point as long as the underlying principle is present
- Be strict but fair -- surface mentions without substance do NOT earn the point
- Do NOT reward length, politeness, or fluency

A tutor response does not need to satisfy every criterion to be considered acceptable; the rubric documents the full set of principles this student needs, and a single response can only address some of them. Score each criterion independently.


────────────────────────────────
### RUBRIC (variable-length checklist)
────────────────────────────────

Criterion 1: Concise and Neutral Delivery
  Check: The tutor's response must be brief, factual, and emotionally neutral, avoiding any defensive argumentation, lengthy explanations, or emotional pleading.

Criterion 2: Explicit Boundary Setting
  Check: The tutor must explicitly state or enforce a behavioral rule or professional protocol (e.g., 'speak only when allowed', 'use standard complaint channels') in response to the disruption.

Criterion 3: Content-Behavior Separation
  Check: The tutor must acknowledge the academic topic or the student's right to an opinion while simultaneously correcting or rejecting the inappropriate manner or language used to express it.

Criterion 4: Strict Professional Distance
  Check: If the student requests special treatment, extra personal time, or informal resolution, the tutor must deny the request and redirect the student to standard institutional procedures.

Criterion 5: Peer De-escalation
  Check: If peers attempt to intervene, attack, or shame the student, the tutor must actively step in to stop them and maintain sole authority over classroom discipline.
